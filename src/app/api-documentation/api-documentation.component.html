Документација сервиса за транслитерацију речи

Овај АПИ пружа услугу транслитерације речи, омогућавајући корисницима конверзију речи између ћириличног и латиничног писма.

УРЛ на којем се налази АПИ је: http://localhost:8080/api/transliteration

АПИ подржава две тачке приступа:
  • /to-latin
  • /to-cyrilic

Тачке приступа:

/to-latin

Метода: post

Content-Type: applictaion/json

Тело захтева: JSON објекат са следећом структуром:
{{'{'}}
"text": "string"
{{'}'}}

text: Улазни текст на ћирилици које траба транслитеровати у латиницу.


Одговор:

Тело одговора: JSON објекат са следећом структуром:
{{'{'}}
"data": [["string", "string"], ["string"]...]
{{'}'}}

 data: Листа могућих транслитерација улазног текста на латиници. Ако постоји више могућих транслитерација, пружене су као низ.

/to-cyrillic

Метода: post

Content-Type: applictaion/json

Тело захтева: JSON објекат са следећом структуром:
{{'{'}}
"text": "string"
{{'}'}}

text: Улазни текст на латиници које траба транслитеровати у ћирилицу.


Одговор:

Тело одговора: JSON објекат са следећом структуром:
{{'{'}}
"data": [["string", "string"], ["string"]...]
{{'}'}}

 data: Листа могућих транслитерација улазног текста на ћирилици. Ако постоји више могућих транслитерација, пружене су као низ.
