Документација сервиса за транслитерацију речи

Овај АПИ пружа услугу транслитерације речи, омогућавајући корисницима конверзију речи између ћириличног и латиничног писма.
АПИ подржава две тачке приступа:
  • /to-latin
  • /to-cyrilic

Тачке приступа:

/to-latin
Метода: post
Content-Type: applictaion/json
Тело захтева: JSON објекат са следећом структуром:{{'{'}} "text": "string"{{'}'}}
text: Улазни текст на ћирилици које траба транслитеровати у латиницу.
